## ⚙️ Configurações e Instalação

```sh
# clonando o projeto
git clone https://github.com/renovatt/resend-email-route-api.git
```

```sh
# Instalando as dependências
npm install
```

```sh
# Compilar e abrir o programa para desenvolvimento
npm run dev
```

```sh
# Compilar e minificar para produção
npm run build
```
